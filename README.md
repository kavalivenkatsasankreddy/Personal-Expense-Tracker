The Personal Expense Tracker is a Python-based console application that helps users record and manage their daily expenses. The application allows users to add expenses, view all recorded expenses, calculate the total spending, and optionally view category-wise expense totals. All data is stored in a CSV file for persistent storage.

This project is beginner-friendly and demonstrates core Python concepts such as file handling, functions, loops, and conditional statements.
